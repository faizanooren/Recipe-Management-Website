# Recipe-Management-Website
A simple recipe management web application built using raw PHP, HTML, CSS, and MySQL (XAMPP).

Features:
- Authentication (User Login and Signup, Logout)
- Edit Profile
– Homepage with navigation
- Recipes Page – Shows all recipes
- Recipe Details Page – Shows full recipe info
- Ingredients Page – Ingredients list for each recipe
- Rating & Review system for recipes
- Save Recipe / Bookmark feature
- Shopping List (a place to note down ingredients that the user needs to buy) 
- Search recipes by name/ingredient
- Upload recipes
  
Tech Stack:
- PHP (plain)
- HTML & CSS
- MySQL (via XAMPP / phpMyAdmin)
